# 🎮 3D Dodge Game

## Overview

3D Dodge Game is an exciting arcade-style game developed using **Unity Engine** and **C#**. The objective is simple: control the player and dodge incoming obstacles. The game tests the player's reflexes, timing, and concentration in an increasingly challenging environment.

---

## Features

* 🎯 Smooth player movement controls
* 🚧 Randomly generated obstacles
* 🎨 Simple and immersive 3D environment

---

## Technologies Used

* **Game Engine:** Unity
* **Programming Language:** C#
* **IDE:** Visual Studio
* **Version Control:** Git & GitHub

---

## Gameplay

The player controls a character or object that can move left and right across the playing field. Obstacles continuously spawn and move toward the player. The goal is to avoid collisions and survive as long as possible.


---

## Game Mechanics

### Player Movement

The player can move horizontally across the platform using keyboard controls.

### Obstacle Spawning

Obstacles are generated at random positions and intervals, creating unpredictable gameplay.

### Collision Detection

When the player collides with an obstacle, the color of the object changes.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/3D-Dodge-Game.git
```

2. Open Unity Hub.

3. Click **Open Project** and select the project folder.

4. Open the main scene from the **Scenes** folder.

5. Press **Play** to run the game.
